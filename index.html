Crystal Collector

Here's how the app works: There will be four crystals displayed as buttons on the page. The player will be shown a random number at the start of the game. When the player clicks on a crystal, it will add a specific amount of points to the player's total score. The game hide this amount until the player clicks a crystal. When they do click one, we update the player's score counter. The player wins if their total score matches the random number from the beginning of the game. The player loses if their score goes above the random number. The game restarts whenever the player wins or loses. When the game begins again, the player always see a new random number and all the crystals will have four new hidden values. Of course, the user's score (and score counter) will reset to zero.

Word Guess

This app randomly picks a letter, and the user has to guess which letter the app chose. When the player wins, we increase the Wins counter and start the game over again (without refreshing the page). When the player loses, we increase the Losses counter and restart the game without a page refresh (just like when the user wins).

Gifstastic

When the user clicks on a button, the page grab 10 static, non-animated gif images from the GIPHY API and place them on the page. When the user clicks one of the still GIPHY images, the gif animate. If the user clicks the gif again, it stop playing. Under every gif, we display its rating (PG, G, so on). This data is provided by the GIPHY API.
